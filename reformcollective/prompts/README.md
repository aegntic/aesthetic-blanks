# Prompts

Every `PartText` embeds its prompt inline. This folder is the human-readable index for branding passes.

## How to use

1. Open `?design=1` to see part IDs.
2. For each part, run the embedded `prompt` with `[USER_PROJECT]` replaced by the real brief.
3. Prefer the prompt over freewriting — chunk size and tone are intentional.

## Priority order (typical brand pass)

1. Nav + footer identity
2. Home hero manifesto + featured work titles
3. Work grid taglines
4. About capabilities + testimonials
5. Nova mega type + FAQ
6. Contact form labels
7. Case study narratives (as needed)

## Motion notes for agents

When adjusting animation, keep:

- mask duration ~0.85s, ease `[0.22, 1, 0.36, 1]`
- marquee ~28s linear infinite
- stagger ~0.07s between children

Do not reintroduce original site assets or client names from the capture.
