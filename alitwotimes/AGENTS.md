## What this is
A semantic wireframe of a high-end director portfolio.
- Massive display type hero
- Full-bleed video/image parts
- GSAP-style staggered text reveals (via PartMotion)
- Project card grid with media upload zones
- Big impact number
- Multi-territory representation contact page
- Zero original brand assets or marketing copy remain

## Editing rules for agents

1. **Images / Video**  
   Use the built-in drag-and-drop or click-to-upload zones.  
   Never hardcode external URLs from the original site.

2. **Text**  
   Do not invent copy.  
   Execute the exact `prompt` stored on each PartText.  
   The prompt already knows the original chunk size, rhetorical shape, and forbidden generics.

3. **Motion**  
   PartMotion components carry the reverse-engineered stagger / reveal behaviour.  
   Prefer adjusting duration / stagger over rewriting the animation.

4. **Structure**  
   Do not add or remove top-level sections unless the user explicitly requests it.  
   The part tree is the contract.

5. **Design mode**  
   Add `?design=1` (or set `data-design="true"` on <html>) to reveal part IDs as overlays.
