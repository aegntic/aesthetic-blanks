# Prompts

Every `PartText` embeds its prompt inline.

## Brand pass order

1. Nav + theme chrome + footer collab form
2. Home split hero + featured work titles
3. What we build pillars + services list
4. Work index + awards lines
5. About mission/team/values
6. Case study narratives (priority first)

## Motion notes

- Split duration ~0.9s, ease `cubic-bezier(0.2, 0.6, 0.35, 1)`
- Marquee ~32s linear infinite
- Mega titles use large clamp scales; keep thin/regular weight feel

Do not reintroduce original client names, award claims tied to those clients, or Astro media assets from the capture.
